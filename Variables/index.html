Variables

JavaScript variables are named containers for storing data values.

Rules to declare a variable name

  • Variable names must begin with a letter
  • Variable names can also begin with $ and _
  • Empty spaces should not be present in variable names
  • Variable names are case sensitive (y and Y are different variables)
  • Reserved words (like JavaScript keywords) cannot be used as variable names
  • Variable names can contain letters, digits, underscores, and dollar signs.

var keyword

The var statement declares a variable.

Example

In this example, you can see variable declaration and assignment with the var keyword

	var a = 1;         // Declare and assign the value 1 to a
	var b = 2, c = 3;  // Declare and assign the value 2 to b and 3 to c
	var d, e, f;       // Declare variables d, e and f

Variables introduction in ES6

  • ES6 introduced two important new JavaScript keywords: let and const.
  • These two keywords provide Block Scope variables (and constants) in JavaScript.
  • Before ES6, JavaScript had only two types of scope: Global Scope and Function Scope.

let keyword

let is a keyword that is used to declare a block scoped variable.

Example

In this example, you can see variable declaration and assignment with the let keyword

	let a = 1;         // Declare and assign the value 1 to a
	let b = 2, c = 3;  // Declare and assign the value 2 to b and 3 to c
	let d, e, f;       // Declare variables d, e and f

const keyword (Constants)

const keyword defines a constant reference to a value.
Because of this, we cannot change constant primitive values, but we can change the properties of constant objects.

Example

In this example, you can see variable declaration and assignment with the const keyword

	const PI = 3.14159265;
	PI = 3.14;      // This will give an error
	PI = PI + 10;   // This will also give an error
